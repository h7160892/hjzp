### 2. 字体规范

| 用途 | 字体 | 大小 | 粗细 |
|------|------|------|------|
| 标题 | 思源宋体 / Noto Serif SC | 24px / 20px / 16px | Bold |
| 正文 | 思源黑体 / Noto Sans SC | 14px | Regular |
| 小字 | 思源黑体 / Noto Sans SC | 12px | Light |
| 数字 | DIN Alternate | 14px | Medium |
| 大字体模式 | 所有字号 × 1.3 | - | - |

### 3. 圆角规范

| 元素 | 圆角 |
|------|------|
| 按钮 | 8px |
| 卡片 | 12px |
| 弹窗 | 16px |
| 输入框 | 8px |
| 标签 | 20px (pill shape) |
| 图标容器 | 50% (圆形) |

### 4. 间距系统（8px 基准）

| 级别 | 数值 | 用途 |
|------|------|------|
| xs | 4px | 图标与文字间距 |
| sm | 8px | 元素内间距 |
| md | 16px | 区块间距 |
| lg | 24px | 页面边距 |
| xl | 32px | 大区块间距 |
| 2xl | 48px | 页面主间距 |

### 5. 阴影系统

```css
/* 卡片阴影 */
box-shadow: 0 2px 8px rgba(44, 95, 92, 0.08);

/* 悬浮阴影 */
box-shadow: 0 4px 16px rgba(44, 95, 92, 0.12);

/* 弹窗阴影 */
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.16);
```

### 6. 动效规范

| 场景 | 时长 | 缓动 |
|------|------|------|
| 页面切换 | 300ms | cubic-bezier(0.4, 0, 0.2, 1) |
| Hover 反馈 | 150ms | ease-out |
| 弹窗出现 | 250ms | cubic-bezier(0.34, 1.56, 0.64, 1) |
| 列表加载 | 200ms | ease-in-out |
| 按钮点击 | 100ms | ease |

**低配设备降级**：所有动效时长 × 0.5，关闭背景虚化效果。

### 7. 图标风格

- 全套细线条线性图标（stroke-width: 1.5px）
- 国风元素：卷轴、印章、祥云、竹、梅、兰、菊
- 扁平化设计，无填充
- Hover 时带轻微渐变高亮（黛青 → 浅黛）

### 8. 组件库

#### 8.1 按钮

```
[主按钮]  黛青底色 #2C5F5C，白字，8px圆角
[次按钮]  白底，黛青边框1px，黛青字
[危险按钮] 赭石底色 #C45B5B，白字
[幽灵按钮] 透明底，黛青边框，黛青字
[禁用状态] 灰底 #E0E0E0，灰字 #999999
```

#### 8.2 卡片

```
- 白色/月白背景 #E8E4DC
- 12px 圆角
- 柔和阴影
- 内部 padding: 16px
- Hover 时阴影加深
```

#### 8.3 表格（无边框轻量化）

```
- 无边框设计
- 隔行浅底色交替
- 表头：黛青底 #2C5F5C，白字
- 悬浮行：浅黛背景 #4A8B87 (透明度 0.1)
- 列宽可拖拽调整
- 支持列筛选
```

#### 8.4 标签（关系标识）

```
- 圆角 pill 形状
- 浅蓝底 #E3F2FD + 深蓝字：过继
- 浅绿底 #E8F5E9 + 深绿字：入赘
- 浅紫底 #F3E5F5 + 深紫字：兼祧
- 浅橙底 #FFF3E0 + 深橙字：收养
- 浅金底 #FFF8E1 + 深金字：外姓入谱
```

#### 8.5 弹窗

```
- 16px 圆角
- 磨砂虚化背景 (backdrop-filter: blur(8px))
- 主操作按钮在右，次操作在左
- 关闭按钮在右上角
```

#### 8.6 世系树节点

```
- 纤细线条连接（stroke: 1px）
- 圆角矩形节点
- 选中态轻微放大 (scale: 1.05)
- 支持平滑缩放、拖拽平移
- 颜色：未选中=浅墨，选中=黛青，已故=灰色
```

### 10. 无障碍

- 最小字号：14px（正文），可切换大字体模式（×1.3）
- 对比度：WCAG AA 级（≥4.5:1）
- 触控区域：≥44×44px（移动端）
- 高对比度模式：支持开关
- 语音朗读：支持长按文本朗读

### 11. 竖排排版规范（古籍风格）

```
- 背景：护眼浅米白 #F5F0E8
- 字间距：1.5em
- 行间距：2em
- 从右向左阅读
- 去除泛黄复古滤镜
- 优化留白，现代排版
```
