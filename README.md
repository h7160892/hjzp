# 虹乡胡氏安定堂家谱管理系统 V2.0

> **安徽省宿州市泗县草沟镇于韩村虹乡安定堂胡氏支族**
> 家族内部无偿使用，严禁商用，严禁外传

---

## 📖 项目简介

为安定堂胡氏家族编纂一部现代化、规范化、永久保存的家谱。采用 Windows EXE + Android APP + Web 三位一体架构，实现采集、编纂、封谱、归档全流程数字化。

## 🎯 核心特性

### 主编端（Windows EXE）
- ✅ 唯一权威数据源，负责编纂与封谱
- ✅ 新中式极简国风 UI（黛青/米白/浅墨黑）
- ✅ 支持 800 人家族数据
- ✅ 封谱后只读，仅允许追加生/娶/卒/葬
- ✅ 公历/农历/干支一键转换
- ✅ 世系树可视化（缩放/拖拽/导出）
- ✅ 古籍竖排 PDF 导出（A3/A4）
- ✅ PNG/SVG 世系树导出
- ✅ 扫描仪集成（TWAIN/WIA） ~~已移除~~
- ✅ 批量打印
- ✅ 自动备份与一键回滚
- ✅ 离线操作

### 采集端（Android APP + Web）
- ✅ 仅用于信息收集，无编纂权
- ✅ 查阅脱敏公开信息（仅自身及上下两代）
- ✅ 支持过继/入赘/兼祧/收养关系标识
- ✅ 照片/录音素材上传
- ✅ OCR 辅助录入
- ✅ 离线采集，联网同步
- ✅ 生物识别登录（指纹/面容）
- ✅ 禁止截图保护
- ✅ 深色模式 + 大字体模式

### 私有云 API
- ✅ JWT 鉴权 + Token 自动刷新
- ✅ 增量同步（减少流量消耗）
- ✅ 文件存储（照片/录音/扫描件）
- ✅ 全操作审计日志
- ✅ 无 Web 管理后台

## 🚀 快速开始

```powershell
# 1. 克隆项目
git clone <repository-url>
cd HuFamilyGenealogy

# 2. 一键部署
.\scripts\deploy.ps1

# 3. 启动服务
cd publish\Hu.Cloud.API
dotnet Hu.Cloud.API.dll --urls "https://0.0.0.0:5001"

# 4. 启动主编端
cd ..\..\publish\Hu.Editor.WPF
.\安定堂家谱编纂端.exe
```
