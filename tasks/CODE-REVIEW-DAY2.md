# Day 2 代码审查清单

## 编纂组（张工）
- [ ] 人物CRUD是否遵循数据模型规范
- [ ] 批量导入是否处理了异常数据
- [ ] 查重算法是否考虑了别名/错别字
- [ ] 事务处理是否正确（导入失败需回滚）
- [ ] 封谱状态下是否禁止修改已有记录

## 世系树组（李工）
- [ ] 递归查询是否有深度限制（防栈溢出）
- [ ] 虚拟化渲染是否正确实现
- [ ] SVG导出是否包含完整字体嵌入
- [ ] 性能测试：800人数据渲染时间

## APP采集端组（刘工）
- [ ] 表单验证是否在前端+后端双重校验
- [ ] 图片上传是否压缩（≤10MB）
- [ ] 录音上传格式是否统一（MP3/WAV）
- [ ] FlagSecure截图保护是否生效
- [ ] 离线数据本地存储是否加密

## Web采集端组（陈工）
- [ ] Blazor组件是否遵循响应式设计
- [ ] 移动端触摸事件是否正常
- [ ] 深色模式CSS变量是否全覆盖
- [ ] 无障碍测试（键盘导航）

## 私有云API组（孙工）
- [ ] JWT Token是否包含过期时间
- [ ] 文件存储路径是否防遍历攻击
- [ ] CORS是否限制为内网IP
- [ ] 速率限制是否生效（防止刷接口）

## 同步安全组（周工）
- [ ] 冲突检测是否基于版本号
- [ ] 备份文件是否RSA二次加密
- [ ] 恢复流程是否先创建应急备份
- [ ] 增量同步是否只传输变更字段

## UI设计组（吴工）
- [ ] 图标是否统一风格（细线条线性国风）
- [ ] 配色是否严格遵循UI-KIT.md
- [ ] 动效时长是否符合规范（≤300ms）
- [ ] 大字体模式是否适配所有界面

## 测试文档组（郑工）
- [ ] 单元测试是否覆盖核心逻辑
- [ ] 集成测试是否模拟真实网络环境
- [ ] 性能测试是否在4GB RAM设备上执行
- [ ] 用户手册是否图文并茂

## 项目管理组（AI）
- [ ] 代码规范检查是否配置
- [ ] CI/CD流水线是否稳定
- [ ] 每日进度报告是否准时发出
- [ ] 任务看板是否实时更新
